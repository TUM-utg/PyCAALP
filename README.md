# Project Assembly Planning

## About

**Project Assembly Planning** is a tool for assembly process and line planning of manufacturing assemblies.
It utilizes graph based methods to represents an industrial's assemblies sequences and by enforcing engineering or geometrical (optional) constraints, it reduces the complexity of problem. 
Furthermore, a Mixed-Integer Program solver deal with the line planning problem for a fixed number of manufacturing stations, optimizing the station times.
